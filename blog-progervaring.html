Programmeerervaring

Een voordeel voor mij is dat ik al een klein beetje programmeerervaring heb. Op de middelbare school had ik informatica in mijn vakkenpakket en heb ik meerdere praktische opdrachten gedaan. Ik heb een website gebouwd met HTML en CSS, wat ik nu dus ook aan het doen ben. Gelukkig merk ik al gelijk dat ik vooruit ben gegaan, want als ik de code van mijn oude website erbij haal kan ik soms wel gillen. Naast HTML en CSS heb ik ook Lua gebruikt als programmeertaal bij het maken van een spelletje in Pico8. Dit is denk ik het leukste project wat ik ooit heb gedaan.

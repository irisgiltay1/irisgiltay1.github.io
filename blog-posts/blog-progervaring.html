✉ Programmeerervaring

← Terug

Voordat ik met deze opleiding ben begonnen, had ik al wat programmeerervaring opgedaan. Niet veel, maar het is iets.

Op de middelbare school had ik informatica in mijn vakkenpakket en heb ik meerdere praktische opdrachten gedaan. Naast een aantal verslagen, presentaties en posters, zijn er ook projecten geweest waarvoor ik met code moest werken.

Onder andere heb ik een website gebouwd met behulp van HTML en CSS. Deze website was gebaseerd op mijn favoriete band. Gelukkig merk ik al gelijk dat ik bij het maken van deze website een stuk vooruit ben gegaan, want als ik de code van mijn oude website erbij haal, zie ik direct reden om te janken.

Naast HTML en CSS heb ik ook Lua gebruikt als programmeertaal bij het maken van een spelletje in Pico8. Hier heb ik veel tijd in gestopt, maar ook heel veel plezier mee gehad. Mijn game "Quack" kreeg een QR code die rond de school hing, net als enkele andere games die in onze klas waren gemaakt, zodat leerlingen hem op hun mobiel konden spelen.
